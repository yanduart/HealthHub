## 🔄 Sistema de progresso

O histórico salva automaticamente:

- ✅ Quais exercícios foram **concluídos**
- ⏸️ Em qual exercício o treino foi **pausado**
- ⏱️ O **tempo restante** do cronômetro no momento da pausa
- 📊 O **status** do treino: `Não iniciado` · `Em andamento` · `Pausado` · `Concluído`

Ao clicar em **"Continuar treino"** no histórico, o app restaura exatamente o ponto onde você parou.

## 🛠️ Tecnologias

- **HTML5** — estrutura semântica com abas e modal nativo
- **CSS3** — variáveis, glassmorphism, animações, responsivo mobile-first
- **JavaScript Vanilla** — sem frameworks, sem dependências
- **Font Awesome 6.5** — ícones via CDN
- **Google Fonts** — Bebas Neue + Outfit
- **Web Audio API** — beep sonoro nativo ao fim do cronômetro
- **localStorage** — persistência do histórico e progresso

## 📱 Responsividade

| Breakpoint | Layout |
|---|---|
| `> 880px` | Desktop — grid 2 colunas, navbar completa |
| `≤ 768px` | Tablet — colunas reduzidas, texto do player oculto |
| `≤ 480px` | Mobile — timer como bottom sheet, layout em coluna, touch-friendly |
